· 💻 Aplicación web de fitness💻 ·

Proyecto realizado durante el CFGS de Desarrollo de Aplicaciones Web

Descripción

En este proyecto, junto a mi compañero, creamos una aplicación web de fitness para compartir recetas.

La web está hecha con PHP y MySQL, y permite:

  • Iniciar sesión
  • Registrarse
  • Publicar recetas
  • Editar y eliminar nuestras recetas
  • Filtrar recetas por palabra clave
  • Ver recetas de otros usuarios
  • Dar like a las recetas
