· 💻 Aplicación web para el profesorado 💻 ·

Proyecto realizado para el final del CFGS de Desarrollo de Aplicaciones Web

Descripción

En este proyecto, junto a mi compañero, creamos una aplicación web para gestionar los permisos de ir al baño de los alumnos.

La web está hecha con PHP y MySQL, y permite:

  • Iniciar sesión para el profesorado
  • Registro para el profesorado
  • Editar información de perfil y contraseña
  • Buscar alumnos por curso, nombre o permisos
  • Añadir alumnos manualmente o mediante CSV
  • Asignar permisos a los alumnos
